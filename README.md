# Todo App com vanilla JS e Redux

Todo App para a talk de vanilla JS e Redux.

As branches sinalizarão o progresso da talk - a `master` é o início.

---

## Get started

- Clone o projeto
- Instale as dependências (`yarn`)
- Rode o projeto local executando:
```sh
$ yarn start
```
